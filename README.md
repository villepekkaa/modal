# Web- ja hybriditeknologiat mobiiliohjelmoinnissa viikkotehtävä 2

Modal component can be used to display content above enclosing view. Documentation
is available on https://reactnative.dev/docs/modal.

Create a new Expo project displaying a screen which contains Show modal message as
Pressable component. Documentation for Pressable component is available on
https://reactnative.dev/docs/pressable.

If user presses the text, a modal screen opens with following content. User can close
the modal by pressing Close, or on Android by using hardware back button.
